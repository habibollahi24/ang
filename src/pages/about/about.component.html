About This Project

  • Angular 20 – The main frontend framework used
  • Standalone Components – For a modular and scalable architecture
  • Angular Signals – Reactive state management with fine-grained updates
  • Router with Query Params – Used for filtering, search, and pagination
  • Lucide Angular – Icon library for modern and customizable SVG icons
  • DaisyUI – Tailwind-based UI component library
  • Tailwind CSS – Utility-first CSS framework for styling
  • dummyjson.com API – Used for fetching product and category data
  • LocalStorage – To persist theme preferences and cart data
  • Responsive Design – Optimized layout for desktop and mobile screens
